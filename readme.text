Интернет-магазин "baby-toys" беговелов, самокатов, велосипедов и подобного транспорта.

В работе использовала сборку pug+scss+gulp.

Постаралась реализовать компонентный подход.

Классы именовала согласно методологии БЭМ с небольшими отклонениями.

Логика написана на Jquery. Использовала плагины: slick-carousel, magnific-popup, validate.

Реализованы практически все страницы за исключением личного кабинета.
